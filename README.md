# Sorteio FIFA

Web app para sorteio de times em campeonatos de futebol de videogame.

## Como funciona

1. O admin cadastra os nomes dos **participantes**
2. O admin cadastra os **times** que serão disputados
3. A quantidade de participantes e times deve ser igual
4. Clica em **Sortear** — cada participante recebe um time aleatório
5. O resultado é exibido na tela para ser compartilhado via print

## Funcionalidades

- Cadastro livre de participantes e times
- Validação automática de quantidades
- Sorteio aleatório (1 participante = 1 time)
- Resultado otimizado para captura de tela
- Totalmente responsivo (mobile e desktop)
- Sem necessidade de login ou cadastro

## Stack

- React + TypeScript
- Tailwind CSS
- Vite
- GitHub Pages

## Desenvolvimento local

```bash
npm install
npm run dev
```

O deploy é feito automaticamente via GitHub Actions ao fazer push na branch `main`.
